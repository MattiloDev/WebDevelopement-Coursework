About page

Information on the creation of my website

Website Concept:

The main purpose of my website is to calculate a Wilks score for a user. The Wilks Coefficient is a score that is used to rank competitors in powerlifting competitions across any weight and gender.

For more detailed information on how Wilks and Epley are calculated vist the Wilks Information Page

Technical Features:

Information on the code and langueages I have used to create my website

HTML:

As well as basic html such as buttons, text and images I used the form tag to create my wilks calculator form, this allowed me to implement functionality such as using the keyboard only to navigate the form and a clear button to clear the form. I have also made extensive use of classes and ID to both referance objects in JavaScript and style Elements or groups, with a Div, in CSS. My navigation bar is a unordered list modified with CSS. I have also used HTML to implement Media such as the youtube videos on the form guide page and image on the Wilks guide page.

CSS:

CSS is used thoughout my website to implement the clean and functional desgin I intended. I used CSS to create my navigation bar from a unordered list as well as place my banner at the top of my website. I used a MainForm ID to allow the body of my website to have a padding around the edges of the screen whilst ensuring that the navigation bar and Banner fill the entire Screen. I also used dynamic CSS such as on buttons or links in the navigation bar which change colour when they are hovered over

JavaScript

I used JavaScript to process the data from my form and calculate the users Total and Wilks. I nested my Wilks() function within my Total() function as the Wilks function requires the users Total in order to calcuate thier Wilks Score.

Design Choices:

I choose to use a fairly simple grey and white colour scheme along with blue for important information. The muted colours contrasted with the standout blue means the users attention can be directed to Important Information

I used Helvetica which is a sans-serif font for it's clean look which fits in with the design of the website

I choose to use a horizontal navigation bar as I feel it works better than a vertical bar when there are <10 pages on a website
